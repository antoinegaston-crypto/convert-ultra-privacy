Politique de Confidentialité pour ConvertUltra
Dernière mise à jour : 14 mai 2026

Le respect de votre vie privée est une priorité absolue. Cette politique de confidentialité détaille la manière dont l'application ConvertUltra traite vos données.

1. Collecte des données personnelles

L'application ConvertUltra ne collecte, ne stocke et ne transmet aucune donnée personnelle permettant de vous identifier (nom, adresse e-mail, numéro de téléphone, etc.). Aucun compte utilisateur n'est requis pour utiliser l'application.

2. Traitement des fichiers (Vidéo et Audio)

Toutes les opérations de conversion de fichiers sont effectuées localement sur votre appareil.

Vos fichiers multimédias ne sont jamais envoyés vers des serveurs externes.

L'application accède uniquement aux dossiers et fichiers que vous sélectionnez explicitement.

3. Services tiers (Intelligence Artificielle)

Pour les fonctionnalités utilisant l'IA (comme l'intégration de Claude d'Anthropic) :

L'application utilise votre propre clé API personnelle.

Vos requêtes sont envoyées directement aux serveurs du fournisseur (Anthropic).

ConvertUltra n'intercepte pas, ne stocke pas et ne consulte pas le contenu de ces échanges. Nous vous invitons à consulter la politique de confidentialité d'Anthropic pour plus de détails sur leur traitement des données.

4. Conservation des données

Puisque aucune donnée n'est collectée sur nos serveurs, nous ne conservons aucune information vous concernant.

5. Consentement

En utilisant ConvertUltra, vous consentez à cette politique de confidentialité.

6. Contact

Pour toute question relative à cette politique, vous pouvez nous contacter via la page de support officielle sur GitHub.
